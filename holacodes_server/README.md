# Holacodes server

[Go Back](https://github.com/MirageJian/holacodes)

This is server code for holacodes.com.

## How to run

Use sql file to create database. Then run `python server.py`.

## Pip dependencies

* tornado
* pymysql

## Error code illustration
`0` - Everything ok  
`1`, `-1`, `2`, `3` etc. - Something wrong, see the error detail in `errmsg`.
